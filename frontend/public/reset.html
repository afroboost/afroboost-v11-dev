L'app ne s'installe pas ?

Chrome garde en mémoire les anciens essais.
Il faut effacer les données Chrome pour ce site :

  1. Ouvrez Chrome sur votre téléphone
  2. Tapez dans la barre d'adresse :
    chrome://settings/content/all
  3. Cherchez afroboost
  4. Appuyez dessus, puis Supprimer les données
  5. Revenez sur www.afroboost.com
  6. Le bouton "Installer" apparaîtra !
