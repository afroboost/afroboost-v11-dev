- [ ] **Étape 2 : Recopier les variantes dans les metadata**

Là où `metadata` est construit (à côté de `pack_sessions`, `tier` et `quantity`),

puis ajouter `**v226_shipping` aux deux appels.

### Tâche 4 : Paiement unifié côté client

**Fichier :** `frontend/src/App.js` — `v225IsDirectCheckout` (~l.1260),
`startProgressiveCheckout` (~l.4364), `OfferCardSlider`

- [ ] **Étape 1 : Élargir l'aiguillage**

```js
  // V226: tout ce qui est payant part en checkout Stripe, produits physiques et
  // achats audio/video compris. Seules les offres a 0 CHF gardent le formulaire :
  // Stripe refuse un montant nul, c'est une limite de plateforme et non un choix.
  //
  // Verifie avant elargissement : `shippingCost` et `tva` n'entrent dans aucun
  // calcul de total (calculateTotal ne les lit pas), et `isAudio` n'est consomme
  // ni par le backend ni par le dashboard. Aucune donnee exploitee n'est perdue.

    if (!offer) return false;
    return v223UnitPrice(offer) > 0;
  };
```

Conserver le nom `v225IsDirectCheckout` : il est le point de décision unique, consommé
par `handleSelectOffer` **et** par le bouton de la carte. Le renommer imposerait de
toucher les deux et créerait un risque de divergence.

non vides), rendre un `<select>` par dimension **avant** le bouton, avec état local au
composant.

`stopPropagation` sur `onClick` **et** `onChange` de chaque `<select>` : sans lui,
ouvrir le sélecteur déclencherait l'achat.

Si une dimension est proposée et qu'aucune valeur n'est choisie, désactiver le bouton
« Réserver » avec un libellé explicite (« Choisissez une taille »). Sans cela le coach
reçoit une commande sans savoir quoi expédier.

Dans `startProgressiveCheckout`, ajouter à l'objet `payload` :

La signature devient `startProgressiveCheckout(offer, quantity, variants)`. Le
troisième paramètre est optionnel : les appelants existants continuent de fonctionner.

Ne toucher à rien d'autre dans ce payload. L'absence de `customerEmail` est délibérée
et documentée (V224) : envoyé à `""`, Stripe rejette la session et les deux tentatives
